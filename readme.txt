= Privacy Notices =

With the default configuration, this plugin, in itself, does not:

* write any user personal data to the database;
* send any data to external servers;
* use cookies.

With the default configuration, this plugin, in itself, does:

*collect the IP addresses of users;

If you activate certain features in this plugin, the form submitter's personal data, including their IP address, may be sent to the service provider. Thus, confirming the provider's privacy policy is recommended. These features include:

* reCAPTCHA to be added

== Installation ==

1. Upload the entire `dynamic-data` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

Feature to be added: 'Dynamic Data' menu in your WordPress admin panel.
